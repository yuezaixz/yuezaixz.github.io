##uwsgi无法启动，报错mysqld.sock

###可能是mysql服务down了

###可能是服务器内存不足造成
该情况可以在wsgi运行的内存中发现提示

spawned uWSGI master process (pid: 12149) fork(): Cannot allocate memory [core/master_utils.c line 726]

这情况就只能将云服务resize了。
