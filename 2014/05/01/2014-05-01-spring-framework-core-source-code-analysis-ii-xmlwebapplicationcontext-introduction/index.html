#3 XmlWebApplicationContext
XmlWebApplicationContext是开发web应用最常用的上下文环境。

##3.1 XmlWebApplicationContext的特性
XmlWebApplicationContext提供了IoC容器对Web环境ServletAPI的集成，并提供Xml解析器，从而可以从xml文件中获取BeanDefinition数据。
